Bugs and Limitations

As the ASCII Importer GUI has been developed due to an actual need, there are a number of known bugs and limitations that will be described below.

Missing features

The template feature does not work yet. This holds true for both, the template selector at the top of the settings panel and the save functionality at its bottom.

1D datasets not tested

The GUI has not yet been tested thoroughly with 1D datasets.

No input validation

Currently, the edit fields are not validated at all. Therefore, the user might easily break the GUI and/or the following import process by entering wrong or not sensible values.
