Introduction

A rather short introduction to the ASCII Importer GUI.

If in a hurry

Suppose you have some ASCII data that you would like to import into the TA GUI. Preview your data using the ... button in the "Filename" panel on the top and make your way through the "Settings" panel while checking in the "File display" panel for the necessary parameters. Finally press Import on the bottom right, and if everything goes well, you'll have your data imported into the TA GUI afterwards.

To the gentle reader

The ASCII Importer GUI emerged from the need to import nearly arbitrary ASCII data into the TA GUI. That allows to display data from many different sources, as long as they are available as ASCII data, nearly regardless how these files look like.

The idea is to preview an ASCII file in the left "File display" panel and using that information set all the necessary parameters in the Settings panel on the right.

GUI layout

The ASCII Importer GUI window should be fairly intuitive and simple to use. It is divided basically into two parts: the file selection and preview part on the left and the settings for the import in the Settings panel on the right.

More details regarding the settings be found in the respective help topic: Settings.

There is a number of known bugs and limitations of this GUI that might be of importance.

BTW: The button with the questionmark shows this help, as you might have figured out already.

Normal operation

The import process starts with selecting a file to import (and preview) by pressing the ... button in the "Filename" panel on the top. After having selected a file, a preview of this (ASCII) file will be displayed in the "File display" panel below. (Please note that it might take up to a few seconds to display the file and that the file preview may be truncated to speed up).

Now you have a preview of your file and can make your way through the Settings panel.

After you have made all necessary settings, simply click on the Import button in the bottom right to import your data and append them to the TA GUI. If everything worked well, you shall now have you data displayed in the TA GUI.

If you decide to abort the import process, simply hit Esc or click on the Cancel button. This will close the ASCII Importer GUI window.

To close the Figure Caption GUI, simply hit the Import or Cancel buttons on the bottom right, depening on whether you want to import your data or discard the whole process. This will bring you back to the main GUI. Alternatively, you may press Esc or Ctrl+w, both cancelling the import.
