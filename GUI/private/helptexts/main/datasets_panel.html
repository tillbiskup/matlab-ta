Dataset display

The "Datasets" panel consists of controls necessary to handle loaded datasets, their visibility, save, remove, duplicate and combine them.

The controls

A short description of all the controls of the panel.

Invisible datasets

List of all datasets currently loaded to the GUI, but not (yet) set visible.

If you load datasets, they first appear in this list. Toggle between invisible and visible allows you to compare different datasets in an easy and quick way. Whereas in "2D plot" display mode, only one dataset is shown at a time (but see the option "Show only active" below), in all other display modes all datasets set to visible are displayed. Furthermore, you could move datasets to invisible that you are sure of that you don't want to include in accumulation.

You can change the label for each dataset by doubly clicking on it (or hitting Return).

Visible datasets

List of all datasets currently loaded to the GUI and set visible.

Datasets in this list are all displayed in 1D display modes (but see the option "Show only active" below), and are by default accumulated via the ACC GUI window, if you don't make changes in that window. Therefore, careful visual inspection of your datasets prior to further processing and explicit toggling of their visibility is highly recommended. Nevertheless, moving a dataset from visible to invisible allows you to quickly move it back without having it to load again from the file system (especially convenient if you made changes to datasets and are not sure yet whether to save them or not).

You can change the label for each dataset by doubly clicking on it (or hitting Return).

Arrows

To move dataset(s) between invisible and visible, there are four arrow buttons between the two list boxes. The two corresponding buttons with double arrows, ⇓ and ⇑, move all datasets into the other list. The remaining two arrow buttons, ↓ and ↑, move the currently selected dataset, respectively.

Multiple datasets

Operations for multiple datasets. Currently this is Combine, a function that is mainly used with Oxford data, but in general applicable to all data that consist of several files that need to be combined to give a full dataset (i.e., a spectrum).

The "Show only active" checkbox provides you with an easy way to just display the currently active dataset in the main axes. The advantage of this is that you can easily scroll through your multiple datasets without having to move them manually between visible and invisible.

Currently active dataset

A set of operations for the currently highlighted (currently active) visible dataset.

Save saves the currently active dataset.

Save as presents a file selection dialogue to the user that allows to specify a filename to save the dataset to.

Remove removes the current dataset from the GUI (but NOT from the file system). If the dataset was modified (star left of its label), the user gets asked what to do (save and remove, remove, cancel).

Undo will provide means to undo the last changes made to the currently active dataset.

Get info will present an additional GUI window to the user providing access to all sorts of information about the currently active dataset, including the full history of operations performed with the toolbox.

Duplicate copies the currently active dataset. This is particularly useful for 2D datasets if one wants to compare different parts of the dataset (spectra at different times, time traces at different field positions).

Note: Undo doesn't work yet.
