File types

The TA toolbox supports a number of file types. For details see the file "TAload.ini" that accompagnies the TAload function. For a few of these formats, you'll find (hopefully) helpful information below.

Tip

You can set the file type that is selected by default in the "Load panel" in your preferences. Use the configuration panel to access the configuration files.

EP LP920 ASCII

As the Edinburgh Instruments L900 software supports different ways of exporting files to ASCII, here are the settings that will allow you to successfully and automatically load the ASCII export files into the TA toolbox:

  • Use "tab" as separator
  • Include details
  • Use "columns"

Please note that binary files are currently not supported and that there is not much of a chance that this will change in the future.
