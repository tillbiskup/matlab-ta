What's new in the TA toolbox/GUI?

A rather short list of new features for each version.

Basically, at this stage, still the complete toolbox counts as "new". Nevertheless, the following list will grow and give you an impression of what got implemented.

An incomplete list of changes within the last versions, in decreasing order of their implementation date.

v.0.0.38

  • Adjusted TAhlASCIIread to fit new needs.

v.0.0.37

  • New split functionality for datasets: TAsplit and command "split" in GUI CMD.
  • Somewhat fixed bug #160.

v.0.0.36

  • Fixed bug #120.
  • Fixed bug with dlmwrite and empty string - not related to bug #151.

v.0.0.35

  • Added new export option in analysis panel: ASCII 2D.
  • New (flashy) splash window.
  • New function TAexport2D
  • New (internal) function TAexceptionHandling.

v.0.0.34

  • Fixed bug #134.

v.0.0.33

  • Updated help for load panel.
  • Added information about supported file types to builtin help.

v.0.0.32

  • Changed modality of busyWindow after process has finished.
  • Fixed bug with busyWindow (complains of timer object).

v.0.0.31

  • Bug fix with mode setting from edit boxes (cf. #124).

v.0.0.30

  • Bug fix

v.0.0.29

  • New ASCII importer for a special kind of files.

v.0.0.28

  • 1D data export can now handle multiple traces (although currently only saving one trace per file, not all traces in one file).
  • Bug fixes (bugs #104, #105, #106).

v.0.0.27

  • Behaviour of suggesting filenames for saving data and figures now configurable: use filename of dataset (default) or use label.
  • NEW: Figure caption GUI for saving information together with figures.
  • Switch allowing to choose whether to export figure title.
  • Bug fixes (bug #97).

v.0.0.26

  • Help window of main GUI redesigned.
  • New panel: GUI internals
    • CMD history display.
    • Setting message/log levels.
    • GUI window for displaying/editing directories used internally in the GUI.
    • Save/load current GUI state ("snapshot", EXPERIMENTAL).
  • Moved and merged MFE panel into display panel.
  • Label of currently active dataset is displayed as title.
  • Command line in main GUI (with some basic functionality and own help window), including:
    • History display.
    • Running scripts (EXPERIMENTAL).
  • GUI appdata now have a version number.
  • Status display in style of traffic lights (green/yellow/red).
  • GUI modes and key bindings for these modes.
  • opentez: Open GUI and display datasets by double-click in Matlab.
  • Bug fixes (bug #85, #86, #88).

v.0.0.25

  • Bug fixes.

v.0.0.24

  • Added importer GUI for ASCII data allowing for interactive import of nearly arbitrary ASCII data.

v.0.0.23

  • Added importer for ASCII data.
  • Added missing files.

v.0.0.22

  • Compatibility fix of BugReporter with Matlab 8
  • Bug fix with accumulating datasets
  • Added field "buffer" to sample structure in TA toolbox format.

v.0.0.21

  • Advanced status reports, now with log levels (info, warning, error).
  • Bug fix with parsing info files.
  • Added configuration options for positioning of main window.
  • Bug fix with exporting to glotaran.
  • Changed font in status window to monospaced on all platforms.

v.0.0.20

  • New key bindings: PgUp and PgDown scroll through datasets.

v.0.0.19

  • Info GUI shows information about single time profiles.
  • Info file parser maps information about single time profiles to toolbox data structure.
  • Extended TA Toolbox data format (v. 0.4)
    • New structure containing information about single time profiles.

v.0.0.18

  • Bug fixes in ACC GUI.
  • Added info window on startup of GUI.
  • Bug fixes with writing info files from Info GUI.
  • Updated configuration files.
    Important: You need to manually adjust your configuration files.
  • Additional settings for zero line.
  • Additional settings for legend.

v.0.0.17

  • Scaling (intensity) of datasets
  • Info GUI can edit comment for dataset.
  • Info GUI can write Info files.
  • TA Info files now can handle multiline fields correctly.

v.0.0.16

  • Added field "shotRepetitionRate" both to data structure and info file
    • TA toolbox data structure: v.0.3
    • Info file format: v.0.2d (2012-03-31)
  • Automatic loading of info files (for EPASCII files)
  • ACC function handles sample parameters.
  • Advanced handling of paths for loading/saving files.
  • Bug fixes.

v.0.0.15

  • New progress displays for loading/saving/exporting datasets.
  • Redesign of "Help:About" window.
  • Buttons can now launch webbrowser even under Windows.

v.0.0.14

  • Bug fixes

v.0.0.13

  • Replacements in edit fields (where appropriate):
    • comma → dot
    • "end" → numerical value
  • Automatic axis limits work now with all MFE display modes
  • New MFE display mode: "relative MFE".

v.0.0.12

  • Figure and 1D export remember last directory.
  • Algebraic functions on two datasets (experimental).
  • Add MFon data to MFoff data.

v.0.0.11

  • Advanced line marker settings.
  • Threshold can now be applied to all visible datasets at once.
  • Show only active dataset: Easier scrolling through datasets.
  • Buttons rearranged for toggling datasets between visible and invisible.
  • 1D export can now export std.dev. for dataset, if available.

v.0.0.10

  • 1D export
  • Numerous bug fixes

v.0.0.9

  • Installation routine
  • Window with tips at start
  • New config options for GUI start

v.0.0.8

  • Fit window
  • TAfit function

v.0.0.7

  • Updated Toolbox file format
    • Now with different file extension (".taz") and saving the data internally as binary (32bit float).
    • Important: The old formats can still be read.
    • Important bug fix with reading the XML part of the files.
  • ACC window now displays all MFE display modes
  • HTML-based help files in GUI-internal help/documentation

v.0.0.6

  • ACC window

v.0.0.5

  • Main GUI can display standard deviation from AVG
  • AVG calculates standard deviation
  • AVG window now displays all MFE display modes

v.0.0.4

  • MFE window
  • Function to write text files

v.0.0.3

  • Info window
  • Can now read TA Info files (and apply contents to dataset)
  • Minor changes in data structure

v.0.0.2

  • AVG window
  • Saving datasets (as ZIP-compressed XML)
  • Scaling of two datasets that should be combined works (basically)
  • Combining datasets works

v.0.0.1

  • Data import improved: Now supports combine (for those formats supporting combining files)
  • Minor changes in data structure
  • Basic Configuration via "Configuration" panel
  • Figure export supports different (configurable) formats (i.e. paper sizes)
  • Help in MATLAB(r) Help Browser
